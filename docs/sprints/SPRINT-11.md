## 목표

## 작업 목록

### 1. MVP 점수 계산 엔진
- [ ] `src/lib/mvp-scoring.ts`
  ```
  MVP 점수 = 자동(70%) + 투표(30%)
  자동: 조회×1 + 응원×3 + 댓글×2 + 프로필유입×2
  투표: 1표 = 10점
  70:30 비율로 정규화하여 합산
  ```
- [ ] MVP 등급 자동 부여
  ```
  1회→루키(rookie), 3회→에이스(ace), 5회→올스타(allstar), 10회→레전드(legend)
  ```
