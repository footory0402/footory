# Sprint 15: 온보딩

> 예상 소요: 2시간
> 선행 조건: Sprint 03 (Auth)
> **v1.1 변경: 팀 선택 스텝 + 홈 넛지**

## 목표
신규 유저 온보딩 (30초) + 첫 5분 시나리오

## 작업 목록

### 1. 온보딩 플로우
- [ ] 카카오 로그인 후 3스텝 온보딩
  - Step 1: 선수/부모 선택
  - Step 2: 이름, 포지션, 출생연도
  - **Step 3: 팀 검색/직접 입력/건너뛰기 (v1.1)**
- [ ] 프사 등록 유도 ("프로필 사진을 올리면 Lv.2!")

### 2. 홈 넛지 (v1.1)
- [ ] 피드 인라인 배너: "첫 영상을 올려보세요!"
- [ ] MVP 안내: "이번 주 MVP에 투표해보세요!"
- [ ] 첫 업로드 완료 시 넛지 숨김

### 3. 핸들 설정
- [ ] 실시간 중복 체크
- [ ] 영문+숫자, 3~20자

## 완료 기준
- [ ] 신규 유저가 30초 안에 온보딩을 완료한다
- [ ] 온보딩 직후 홈에 콘텐츠가 보인다
- [ ] 팀 선택 스텝이 동작한다
- [ ] 넛지 배너가 표시되고, 업로드 후 사라진다
