# SPRINT-29: 업로드 위저드 구조 개편 (5단계 → 3단계)

> Phase F (영상 UX 개편) | 의존: SPRINT-27 완료
> 예상: 5~6시간
> 우선순위: 2위 (4개 팀 전원 합의)

---

## 목표

5단계 위저드를 3단계로 축소하고, 부모 전용 간편 모드를 추가.
선택적 기능(스팟라이트/슬로모/효과)은 "꾸미기" 탭 UI로 통합.

---

## 현재 구조 (as-is)

```
Step 0: 파일 선택 (+ 부모: 자녀 선택)
Step 1: 구간 트리밍 (필수)
Step 2: 주인공 표시 (선택 — 건너뛰기)
Step 3: 슬로모션 (선택 — 건너뛰기)
Step 4: 스킬라벨 + 효과 + BGM + 태그메모 (과적재)
Step 5: 확인 프리뷰
```

## 개편 구조 (to-be)

```
Step 1: 파일 선택 + 구간 자르기
  - 파일 선택 → 자동으로 트리머 표시
  - 부모: ChildSelector 상단 표시
  - 트림 완료 시 "다음" 활성화

Step 2: 꾸미기 (탭 UI, 모두 선택사항)
  ┌─────────┬─────────┬─────────┬─────────┐
  │ 나 찾기  │ 느린재생 │  효과   │  BGM    │
  └─────────┴─────────┴─────────┴─────────┘
  - 각 탭은 독립적, 순서 무관
  - 아무것도 안 해도 "다음" 가능
  - 상단에 축소 비디오 프리뷰 고정
  - "간편 업로드" 버튼: 꾸미기 전체 건너뛰기

Step 3: 태그 + 확인
  - 스킬 라벨 선택 (필수 1개)
  - 태그 + 메모
  - 설정 요약 카드
  - "업로드" 버튼
```

## 부모 간편 모드

```
Step 1: 자녀 선택 + 파일 선택 + 구간 자르기
Step 2: 태그 + 확인 (스킬 라벨 선택사항)
→ 꾸미기 단계 자체를 생략 (나중에 아이가 수정 가능)
```
