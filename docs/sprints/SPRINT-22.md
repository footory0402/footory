### 2. 부모 바텀탭

파일: `src/components/layout/BottomTab.tsx` 수정 (role='parent' 분기)

```
🏠 홈(자녀 대시보드) / 💬 DM / ⚙️ 설정
/                    /dm     /settings
```

3탭. MVP/탐색/팀 탭은 부모에게 불필요.

### 3. 자녀 대신 업로드

파일: `src/components/parent/ProxyUpload.tsx`

```
자녀 대시보드 > [영상 올려주기]

플로우:
  1. 영상 선택 (갤러리)
  2. 태그 선택 (1v1돌파, 슈팅, 패스...)
  3. 한 줄 설명 (선택)
  4. [올리기]

업로드 시:
  clips.profile_id = 자녀의 profile_id
  clips.uploaded_by = 부모의 profile_id (새 컬럼)
  → 자녀 프로필에 영상이 표시됨
  → 자녀에게 알림: "보호자님이 영상을 올려줬어요!"
```

DB 변경:
```sql
ALTER TABLE clips ADD COLUMN uploaded_by UUID REFERENCES profiles(id);
-- NULL이면 본인 업로드, 값이 있으면 대신 업로드
```
