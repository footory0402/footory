# Sprint 09: 피드 시스템

> 예상 소요: 3~4시간
> 선행 조건: Sprint 03 (DB), Sprint 02 (컴포넌트)
> **v1.1 변경: 팔로우 피드 → 추천 기반 피드 전면 교체**

## 목표
홈 탭에 추천 기반 피드 + 이번 주 베스트 캐러셀 구현

## 작업 목록

### 1. 추천 피드 알고리즘
- [ ] `src/lib/feed-algorithm.ts`
  ```
  1순위: 팔로우한 선수의 새 콘텐츠
  2순위: 같은 팀 동료
  3순위: 같은 지역 + 연령대
  4순위: 같은 포지션
  5순위: 전체에서 응원 많은 순
  ```
- [ ] 팔로우 수에 따른 비율 조절
  - 0명: 100% 추천 / 1~5명: 30%팔로우+70%추천 / 6~20명: 50:50 / 21+: 70:30

### 2. 이번 주 베스트 캐러셀
- [ ] `src/components/feed/BestCarousel.tsx`
  - 응원+조회 합산 높은 순
  - 가로 스크롤, 세로 썸네일 (3:4)
  - 선수명 + 태그 오버레이
  - 탭하면 영상 재생

### 3. 피드 카드 개선
- [ ] `src/components/feed/FeedCard.tsx`
  - 태그 + 짧은 설명 표시 (예: "1v1 돌파 | 수비 2명을 제치는 장면")
  - 기존 "새 하이라이트를 등록했어요" 문구 대체
  - 선수 이름 탭 → 프로필 이동

### 4. 신규 유저 넛지
- [ ] 피드 3번째 위치에 인라인 배너
  - "🎬 첫 영상을 올려보세요! MVP 투표 후보에 자동 등록"
  - [영상 올리기] CTA 버튼
  - 첫 업로드 완료 시 숨김

### 5. 피드 아이템 자동 생성 (기존)
- [ ] 6종 자동 게시: highlight, featured_change, medal, stat, season, top_clip
- [ ] 응원(Kudos) 기능
- [ ] 댓글 기능
