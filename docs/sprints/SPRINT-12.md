# Sprint 12: 팀 기능

> 예상 소요: 2~3시간
> 선행 조건: Sprint 03, Sprint 04
> **v1.1 변경: 현재/이전 소속 분리 + alumni 상태**

## 목표
팀 생성/가입/관리 + 현재/이전 소속 분리 + alumni 권한

## 작업 목록

### 1. 팀 기본
- [ ] 팀 생성 (누구나) — 이름, 로고, 설명, 지역
- [ ] 초대코드 가입 (8자 hex)
- [ ] 팀 프로필 (로고/소개/멤버)
- [ ] 팀 앨범 (사진/영상 업로드)
- [ ] 앨범→내 프로필 가져오기

### 2. 현재/이전 소속 분리 (v1.1)
- [ ] 팀 탭 메인에서 현재/이전 분리 표시
  ```
  현재 소속
    🏟 수원FC U-15 (강조)
  이전 소속 (회색)
    🏟 강남FC U-14 [졸업]
  ```
- [ ] `AlumniLabel` 컴포넌트 — "(졸업)" 회색 라벨

### 3. Alumni 권한 (v1.1)
- [ ] alumni 멤버:
  - 앨범 열람: O
  - 내 프로필로 가져오기: O (자기 영상만)
  - 새 영상 업로드: X (버튼 숨김)
  - 팀 정보 수정: X
  - 멤버 리스트에서 "(졸업)" 표시

### 4. 팀 랭킹 연동 (v1.1)
- [ ] 팀 프로필에 활동 점수 + 순위 표시
- [ ] MVP 배출 선수 하이라이트

### 5. 팀 공개 프로필
- [ ] `/t/{handle}` 공개 접근 가능

## 완료 기준
- [ ] 팀 탭에서 현재/이전 소속이 분리된다
- [ ] alumni 멤버는 업로드/수정이 불가하다
- [ ] alumni도 앨범 열람 + 자기 영상 가져오기는 가능하다
- [ ] 팀 프로필에 활동 점수가 표시된다
