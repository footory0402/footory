# SPRINT-26: 최종 QA + 폴리싱

> Phase E | 의존: SPRINT-22 완료 (모든 스프린트 완료 후)
> 예상: 3~4시간

---

## 목표

전체 역할별 시나리오 테스트 + 디자인 검증 + 에러/빈 상태 + 성능 최적화

---

## 시나리오 테스트

### 1. 신규 선수 시나리오

```
1. 카카오 로그인
2. 역할 선택: ⚽ 선수
3. 온보딩: 이름/포지션/출생연도/키/몸무게/선호발
4. 팀 가입: 팀 코드 입력
5. 홈 진입: 추천 피드 + 챌린지 배너 + 초보자 퀘스트
6. 🔍 탭: 검색+탐색 오버레이 열림, 랭킹 표시
7. 첫 영상 업로드 → 푸시 권한 요청 → 퀘스트 체크
8. 타임라인에 "첫 영상 업로드" 이벤트 생성
9. 피드에 내 카드 표시
10. 다른 선수 팔로우 → 퀘스트 체크
11. 응원 보내기 → 이모지 리액션 5종 동작
12. DM 보내기 → 퀘스트 체크
13. MVP 투표 → 퀘스트 체크
14. 프로필 확인: 기본 정보, 영상, 스킬 태그
15. 프로필 PDF 내보내기
16. 공개 프로필 링크 공유 → OG 이미지 확인
```

### 2. 부모 시나리오

```
1. 카카오 로그인
2. 역할 선택: 👨‍👩‍👦 부모
3. 자녀 연결 (검색 or 초대)
4. 홈: 자녀 대시보드 표시
5. 바텀탭: 3탭 (홈/DM/설정) 확인
6. 자녀 대신 영상 업로드
7. 자녀에게 알림 확인
8. 주간 리캡 카드 확인
9. 코치에게 DM 보내기
10. 자녀 프로필 PDF 내보내기
11. 알림 설정 (ON/OFF, 조용한 시간)
```

### 3. 코치 시나리오

```
1. 카카오 로그인
2. 역할 선택: 📋 코치
3. 인증 요청 (팀 코드 or 증빙)
4. 인증 완료 → ✅ 뱃지
5. 팀 생성 + 코드 공유
6. 선수 영상에 코치 리뷰 남기기
7. 비공개 피드백 확인 (선수만 보이는지)
8. 선수에게 DM 보내기
9. 부모에게서 DM 수신
10. 바텀탭: 5탭 (선수와 동일)
```

### 4. 스카우터 시나리오

```
1. 코치로 가입 + 인증
2. 🔍 검색으로 선수 탐색
3. 관심 선수 추가 (⭐)
4. 워치리스트 확인 (비공개)
5. 관심 선수에 메모 추가
6. 관심 선수 새 영상 시 알림 확인
7. 선수에게 DM 보내기
```

### 5. DM 시나리오

```
1. 팔로잉 선수에게 DM → 바로 대화
2. 비팔로우 선수에게 DM → DM 요청
3. 상대가 수락 → 대화 시작
4. 상대가 거절 → 거절 표시
5. 영상 클립 공유
6. 읽음 표시 ✓✓
7. 차단 → 상대 콘텐츠 숨김
8. 신고 → 카테고리 선택 후 제출
9. 💬 탭 뱃지 카운트
```

## 디자인 토큰 검증

```
모든 화면에서 확인:
  □ 배경: #0C0C0E
  □ 카드: #161618
  □ 카드alt: #1E1E22
  □ 골드: #D4A853
  □ 텍스트1: #FAFAFA
  □ 텍스트2: #A1A1AA
  □ 텍스트3: #71717A
  □ 본문 폰트: Noto Sans KR
  □ 숫자/스탯: Oswald
  □ 카드 radius: 10~14px
  □ 바텀탭: 54px, 블러, 활성=골드
  □ 모바일 뷰포트: 430px 기준
```

## 에러/빈 상태 처리

```
각 화면에서 확인:
  □ DM 목록 비어있을 때: "메시지가 없어요" + CTA
  □ 알림 비어있을 때: "알림이 없어요"
  □ 챌린지 없을 때: 배너 숨김
  □ 퀘스트 완료 시: 숨김 또는 완료 표시
  □ 워치리스트 비어있을 때: 안내 문구
  □ 자녀 미연결 시: 연결 유도
  □ 검색 결과 없을 때: "검색 결과가 없어요"
  □ 네트워크 에러: 재시도 버튼
  □ 로딩 상태: 스켈레톤 UI
```

## 성능 최적화

```
□ DM Realtime 구독: 컴포넌트 언마운트 시 해제
□ 알림 폴링 → Realtime으로 전환
□ 이미지 lazy loading (프로필 사진, 썸네일)
□ 대화 목록: 가상화 (10개 이상 시)
□ 검색 자동완성: 디바운스 300ms
□ 타임라인: 무한 스크롤 (20개씩)
□ PDF 생성: 로딩 인디케이터
```
