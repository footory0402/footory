# SPRINT-11c: 팀 미니멀 리팩토링

> Phase A | 의존: SPRINT-11b-v2 완료 (Phase A 마지막)
> 예상: 2~3시간

---

## 목표

팀을 "관리 도구"에서 "소속 라벨 + 자동 팀 피드"로 경량화.
팀 앨범 제거, 멤버 영상 자동 피드, 팀 코드 공유.

---

## 배경

> 워크숍 결정: Footory는 팀 관리 앱이 아니라 개인 선수 중심 앱.
> 팀은 5순위. "소속 라벨"이지 "관리 단위"가 아님.
> Hudl/TeamSnap처럼 B2B 제휴가 아니라, 선수 1명이 팀 만들고
> 카톡에 코드 공유하면 10~20명이 가입하는 C2C 바이럴.

---

## 작업

### 1. 팀 앨범 → 팀 피드 전환

기존: team_albums 테이블에 별도 업로드
변경: 팀 멤버들의 clips를 team_id로 필터링 = "팀 피드"

```
팀 상세 화면:
┌─────────────────────────────────┐
│ ← 수원FC U-15         ⚙️       │
├─────────────────────────────────┤
│                                 │
│ 🏟 수원FC U-15                  │
│ @suwon-fc-u15 · 수원 · U-15    │
│ 멤버 12명                       │
│                                 │
│ ── 멤버 ──                      │
│ (사진) 김민준 FW Lv.4           │
│ (사진) 배준혁 FW Lv.3           │
│ (사진) 서현우 MF Lv.3           │
│ 전체 보기 (12명) →              │
│                                 │
│ ── 팀 피드 ──                   │
│ (멤버들의 최신 영상, 자동 수집)   │
│ (FeedCard 컴포넌트 재사용)       │
│                                 │
│ 팀 코드: SUWON-2026  [복사]     │
│                                 │
└─────────────────────────────────┘
```

### 2. 팀 코드 시스템

팀 생성 시 자동 생성: `{팀이름앞4자}-{연도}` 또는 랜덤 6자리
```sql
ALTER TABLE teams ADD COLUMN invite_code TEXT UNIQUE;
-- 생성 시 자동 부여, 예: "SUWON-2026" 또는 "ABC123"
```

팀 코드 공유:
```
[복사] 버튼 → 클립보드에 복사
[카톡 공유] → "수원FC U-15에 참여하세요! 코드: SUWON-2026
             Footory에서 입력하세요 → footory.app"
```

팀 가입:
```
팀 탭 > [가입] > 팀 코드 입력 > 즉시 가입 (승인 불필요)
→ team_members INSERT (role: 'member')
```

### 3. 팀 관리 최소화

```
팀 관리자(admin)가 할 수 있는 것:
  - 팀 정보 수정 (이름, 연령대, 지역)
  - 멤버 강퇴
  - 팀 코드 리셋

팀 관리자가 할 수 없는 것 (제거):
  - 팀 앨범 관리 (삭제됨)
  - 팀 공지 (v1.5 그룹 DM으로 대체 예정)
```

### 4. 팀 앨범 관련 코드/UI 제거

```
제거 대상:
  - team_albums 테이블 사용 → 대체: clips WHERE profile_id IN (팀 멤버)
  - TeamAlbum 컴포넌트 → 제거
  - 팀 앨범 업로드 UI → 제거
  - 팀 상세 > 앨범 탭 → "팀 피드"로 교체

유지:
  - teams 테이블
  - team_members 테이블 (admin/member/alumni)
  - 팀 랭킹 (team_ranking_cache)
  - alumni 상태
```

### 5. 팀 피드 쿼리

```sql
-- 팀 피드: 현재 멤버(member)의 최신 영상
SELECT c.* FROM clips c
JOIN team_members tm ON c.profile_id = tm.profile_id
WHERE tm.team_id = {team_id}
  AND tm.role IN ('admin', 'member')  -- alumni 제외
ORDER BY c.created_at DESC
LIMIT 20;
```

### 6. 팀 생성 간소화

```
팀 탭 > [+ 팀 만들기]
  - 팀 이름 (필수)
  - 연령대 (U-12/U-13/U-14/U-15/U-18)
  - 지역 (시/도 선택)
  - [만들기]
  → 팀 코드 자동 생성
  → "팀 코드를 공유하세요!" + [카톡 공유] [복사]
```
