## 1. 네비게이션 변경

### 바텀탭 (v1.1 → v1.2)

```
v1.1: 🏠홈    🏆MVP    🔍탐색    👤프로필    👥팀
v1.2: 🏠홈    🏆MVP    💬DM     👤프로필    👥팀

라우트:
  /         → 홈 (추천 피드)
  /mvp      → MVP (주간 투표)
  /dm       → DM (대화 목록)    ← 신규
  /profile  → 프로필
  /team     → 팀
```

### 헤더 변경

```
v1.1: FOOTORY                    🔔
v1.2: FOOTORY               🔍  🔔

🔍 탭 → 검색+탐색 오버레이 (풀스크린)
🔔 탭 → 알림 센터
```

### 검색+탐색 오버레이

기존 /explore 페이지의 모든 콘텐츠가 오버레이로 이동:
- 검색바 (자동 포커스)
- 필터 탭 [전체] [선수] [팀] [태그]
- 인기 선수 랭킹
- 떠오르는 선수 캐러셀
- 팀 랭킹 캐러셀
- 태그별 인기 클립

오버레이는 /explore 라우트가 아닌, 모달/시트 형태.
뒤로가기 또는 X로 닫으면 이전 화면 복귀.

## 2. 계정 관리

### 2.1 역할 3종

```
profiles.role:
  'player'  — 선수 (기본값)
  'parent'  — 부모/보호자
  'coach'   — 코치/스카우터

profiles.is_verified:
  false — 기본
  true  — 코치/스카우터 인증 완료
```

### 2.2 온보딩 플로우

**공통**: 카카오 로그인 → 역할 선택 화면

**선수**: 역할 선택 → 이름/포지션/출생연도 → 팀 가입(선택) → 프사(선택) → 홈

**부모**: 역할 선택 → 자녀 연결(검색/초대/나중에) → 완료 → 자녀 대시보드

**코치/스카우터**: 역할 선택 → 이름/소속/역할상세 → 인증 요청(선택) → 팀 관리(선택) → 완료

### 2.3 코치 인증 방법 (3가지)

1. 팀 초대 코드 입력 → 팀 admin 확인 → 즉시 인증
2. 소속 증빙 제출 (자격증/명함) → 운영팀 검토 (24~48h)
3. 기존 인증 코치 추천 → 즉시 인증

### 2.4 역할별 권한

```
                     선수    부모    코치(인증)  스카우터(인증)
영상 업로드(본인)     ✅      —       ❌          ❌
영상 업로드(자녀)     —       ✅      —           —
MVP 투표             ✅      ❌      ❌          ❌
응원/댓글            ✅      ✅      ✅          ❌
팔로우               ✅      ❌      ❌          ❌
DM                   ✅→선수  ✅→코치 ✅→누구나   ✅→선수/부모
코치 태그            ❌      ❌      ✅          ❌
관심 선수 리스트     ❌      ❌      ❌          ✅
```

### 2.5 부모 전용 UI

- 바텀탭 3탭: 🏠홈(자녀 대시보드) / 💬DM / ⚙️설정
- 홈: 자녀 주간 활동 요약 + 최근 활동 + 팀 소식
- 자녀 대신 업로드 기능
- 프로필 PDF 내보내기 (자녀)

## 3. 알림 시스템

### 3.1 알림 센터

- 단순 시간순 리스트 (시간 구분 헤더 없음)
- 응원만 그룹핑 ("외 N명")
- MVP 알림만 골드 카드 스타일
- 탭하면 해당 화면으로 이동 (인라인 액션 없음)

### 3.2 알림 설정 (사용자 노출)

- 🔔 알림 ON/OFF (마스터 토글)
- 🌙 조용한 시간 (기본 22:00~08:00)
- 고급 설정: 프로필 > 설정 > 알림 상세 (숨겨진 메뉴)

### 3.3 알림 우선순위 (내부)

```
🔴 무조건 (끌 수 없음): mvp_win, team_invite, dm
🟡 기본 ON: kudos, comment, follow, vote_open, mvp_result, level_up
🟢 기본 OFF: upload_nudge, vote_remind
```

### 3.4 푸시

- FCM 웹 푸시 (Service Worker)
- 권한 요청 타이밍: 첫 영상 업로드 직후
- 알림톡 2종만: MVP 선정(선수), 주간 리캡(부모)

## 4. DM

### 4.1 범위

- v1.2: 1:1 텍스트 + 클립 공유 + 읽음 표시
- v1.5: 그룹 DM, 이미지 첨부

### 4.2 DM 권한

```
팔로잉 O → 바로 DM 가능
같은 팀 → 팔로우 없이 DM 가능
그 외 → DM 요청 (수락 후 대화)
코치/스카우터(미인증) → 미성년에게 DM 불가
```

### 4.3 안전

- 차단: 상대 DM/응원/댓글/프로필접근 전부 차단, 상대에게 미노출
- 신고: 카테고리(괴롭힘/스팸/부적절/기타), 24h 내 검토
- 누적 3회 신고: DM 7일 정지

## 7. 재미 요소

### 7.1 주간 챌린지 (challenges)

매주 월요일 새 챌린지. 챌린지 태그로 업로드 → 응원 많은 순 랭킹 → 1위 뱃지.
홈 피드 상단에 챌린지 배너.

### 7.2 퀘스트 (quest_progress)

초보자 퀘스트: 프사 등록, 첫 영상, 첫 팔로우, 첫 응원, 첫 DM → "신입생 뱃지"
주간 퀘스트: 영상 1개, 투표 3표, 응원 5개 → 보너스 XP

### 7.3 프로필 방문자

profile_views 카운트. 알림: "👀 N명이 프로필을 봤어요" (숫자만, 누구인지 X).
프로필에 "이번 주 조회 N회" 표시.

## 8. 코치/스카우터 도구

### 8.1 코치 태그 (coach_reviews)

인증 코치만. 자기 팀 + 팔로우한 선수 영상에:
- 한 줄 코멘트 (공개, 80자)
- 스킬 평가: good/great/excellent
- 비공개 피드백 (선수에게만)
영상에 "📋 코치 리뷰" 뱃지 표시.

### 8.2 스카우터 워치리스트 (scout_watchlist)

인증 스카우터만. 관심 선수 추가 (비공개).
메모 기능. 관심 선수 새 영상 시 알림.

## 9. 팀 미니멀 재설계

- 팀 앨범 삭제 → 멤버 영상이 "팀 피드"로 자동 노출
- 팀 = 소속 라벨 (관리 단위 아님)
- 팀 코드로 가입 (카톡 공유)
- 팀 화면: 멤버 리스트 + 팀 피드 + 팀 코드
- alumni 상태 유지
