### v1.3 핵심
두 축으로 진행:

**축 1 — 안정화**: UX 폴리시 → v1.2 검증 → 핵심 플로우 → 14세 동의
**축 2 — 영상 엔진**: 업로드 5스텝 편집 + Cloudflare Containers FFmpeg 렌더링

### 영상 엔진 요약
유저가 영상 업로드 시 5스텝 편집 플로우:
1. 트리밍 (구간 자르기)
2. 주인공 표시 (터치 → EA FC 골드 원형 링, 2초)
3. 슬로모션 (구간 지정 → 0.5x 리플레이 자동 생성)
4. 태그+BGM+효과 (스킬 라벨, 배경음악, 색보정/레터박스/EA FC 카드)
5. 프리뷰 → 업로드

서버에서 FFmpeg 7-Pass로 렌더링:
  Pass 1: 색보정+레터박스
  Pass 2: 텍스트 오버레이
  Pass 3: 골드 링 합성
  Pass 4: 슬로모션 리플레이
  Pass 5: 인트로/아웃트로
  Pass 6: 합본
  Pass 7: BGM 믹싱

결과물을 R2에 저장, Supabase Realtime으로 완료 알림.

### 기술 스택 (신규)
- Cloudflare Containers: FFmpeg Docker 컨테이너 (Worker가 라우팅)
- R2 바인딩: 네이티브 연동 (presigned URL로 직접 업로드)
- 스케일 투 제로: 렌더링 요청 없으면 비용 0
