### 0.1 바텀 탭 네비게이션 (5탭)

```
┌──────────────────────────────────────────────────────┐
│  🏠 홈      🏆 MVP      🔍 탐색      👤 프로필    👥 팀  │
│ (피드)    (주간MVP)   (랭킹+검색)   (내 프로필)  (팀허브) │
└──────────────────────────────────────────────────────┘
```

| 탭 | 라우트 | 역할 | 사용 빈도 |
|----|--------|------|----------|
| 🏠 홈 | `/` | 추천 피드 + 이번 주 베스트 캐러셀 | 매일 여러 번 |
| 🏆 MVP | `/mvp` | 주간 MVP 투표 + 순위 + 아카이브 + 보상 | 주 2~3회 |
| 🔍 탐색 | `/discover` | 선수·팀 랭킹 + 검색 + 태그별 그리드 + 팔로우 | 주 1~2회 |
| 👤 프로필 | `/profile` | 내 프로필 관리 + 영상 업로드 (앱의 심장) | 주 2~3회 |
| 👥 팀 | `/team` | 팀 관리 + 앨범 + 현재/이전 소속 분리 | 주 1회 |

### 0.2 홈 — 추천 피드

**피드 알고리즘 (팔로우 없어도 콘텐츠 보임):**
```
1순위: 팔로우한 선수의 새 콘텐츠
2순위: 같은 팀 동료의 콘텐츠
3순위: 같은 지역 + 같은 연령대
4순위: 같은 포지션
5순위: 전체에서 응원 많은 순
```

**팔로우 수에 따른 피드 비율:**
- 0명: 100% 추천
- 1~5명: 30% 팔로우 / 70% 추천
- 6~20명: 50% / 50%
- 21명+: 70% / 30%

**상단 "이번 주 베스트" 캐러셀:**
- 응원+조회 합산 높은 순
- 가로 캐러셀, 세로 썸네일 (3:4)
- 탭하면 영상 재생

### 0.3 MVP 탭 — 주간 투표 + 보상

**MVP 점수 = 자동(70%) + 투표(30%)**

자동 점수:
- 조회수 × 1점
- 응원(Kudos) × 3점
- 댓글 × 2점
- 프로필 방문 유입 × 2점

투표 점수:
- 1표 = 10점
- 투표 기간: 토요일 09:00 ~ 일요일 23:59
- 1인당 주 3표 (다른 선수에게만, 본인 불가)
- 같은 영상에 중복 투표 불가

**주간 사이클:**
```
월 00:00 → 새 주차 시작 + 이전 주 MVP 확정 알림
월~금    → 영상 업로드. 자동 점수 실시간 집계. 순위 변동.
토 09:00 → 투표 OPEN. 푸시 알림.
일 23:59 → 투표 마감.
월 00:00 → 최종 순위 확정. 1위 MVP 알림. 새 주차 시작.
```

**MVP 보상 체계:**
```
즉시: 프로필에 "🏆 이번 주 MVP" 뱃지 (1주간)
      피드에 MVP 카드 자동 게시 (골드 테두리)
      하이라이트에 "🏆 MVP" 라벨 영구 표시

누적:
  MVP 1회 → ⭐ MVP 루키
  MVP 3회 → 🌟 MVP 에이스
  MVP 5회 → 👑 MVP 올스타
  MVP 10회 → 💎 MVP 레전드

미래 (v2+):
  월간 MVP → 프리미엄 뱃지
  시즌 MVP → 파트너 브랜드 쿠폰
  MVP 리더보드 상위 → 스카우터 노출 우선순위
```

### 0.4 탐색 탭 — 랭킹 + 검색 + 발견

**상단 필터 탭:** [전체] [선수] [팀] [태그]

**선수 랭킹 (인기 점수):**
```
총 응원 × 3 + 총 조회 × 1 + 총 댓글 × 2 + 팔로워 × 5 + MVP횟수 × 50 + 최근7일 업로드 × 10
```
정렬: 인기순 / 팔로워순 / MVP횟수순 / 최신순
필터: 포지션 / 지역 / 연령대

**팀 랭킹 (활동 점수):**
```
소속 선수 업로드수 × 5 + 소속 선수 응원수 × 2 + MVP배출수 × 30 + 팀앨범 영상수 × 3 + 팀팔로워수 × 5
```

**검색:** 상단 검색바 → 탭하면 검색 오버레이 (선수/팀/@핸들/태그)

**태그별 그리드:** 2~3열 썸네일 그리드 (인스타 탐색 스타일)

### 0.5 팔로우 프로세스

**8개 진입점:**
1. 탐색 > 선수 랭킹 [팔로우]
2. 탐색 > 떠오르는 선수 캐러셀 [팔로우]
3. 피드 카드 > 프로필 이동 > [팔로우]
4. 타인 프로필 > [팔로우]
5. 팀 상세 > 멤버 리스트 [팔로우]
6. MVP 탭 > 순위 카드 [팔로우]
7. 검색 결과 [팔로우]
8. 공개 프로필 /p/{handle} [팔로우]

**팔로우 추천:**
1순위: 같은 팀 동료 (미팔로우)
2순위: 친구의 친구
3순위: 같은 지역 + 포지션
4순위: 같은 연령대 인기 선수
5순위: 이번 주 MVP 후보

### 0.6 팀 현재/이전 소속

**team_members.role:** admin / member / alumni

**alumni 권한:**
- 앨범 열람: O
- 내 프로필로 가져오기: O (자기 영상만)
- 새 영상 업로드: X
- 팀 정보 수정: X

**시즌 이력:**
- seasons.is_current (현재 소속 표시)
- 팀 변경 시: 이전 시즌 is_current=false, 새 시즌 생성, 이전 팀 role→alumni

### 0.7 신규 유저 첫 5분 시나리오

```
1. 카카오 로그인
2. 온보딩 30초: 선수/부모 → 이름/포지션/출생연도 → 팀 선택 → 프사 유도
3. 홈 진입: 추천 피드가 이미 차 있음 (지역/포지션 기반)
4. 넛지: "첫 영상을 올려보세요!" 인라인 배너 (피드 3번째 위치)
5. 첫 업로드 → 하이라이트 자동 → 프로필 반영 → MVP 후보 등록 → 레벨 업
6. 응원 알림 → 재방문
```

### 🎯 [기획팀] 리뷰

**관점: "이 설계가 PRD의 핵심 가설을 검증할 수 있는가?"**

**✅ 잘 된 것:**
- 프로필 OS 컨셉이 네비게이션에 잘 반영됨. 프로필 탭이 앱의 심장.
- "행동이 곧 콘텐츠" (업로드/기록 → 피드 자동 게시) — 콘텐츠 생산 비용 최소화.
- 프로필 완성도 시스템 — Activation 퍼널을 자연스럽게 유도.

**⚠️ 문제점 + 개선:**

| # | 문제 | 개선안 |
|---|------|--------|
| P-01 | 팀이 더보기 하위로 내려감 → GTM 전략과 충돌. 초기 확산은 "팀 단위 도입"인데 팀 접근이 어려움 | **팀을 바텀탭으로 복원**. 4탭: 홈/탐색/프로필/팀. 더보기는 프로필 안 설정으로 이동 (인스타 방식) |
| P-02 | 탐색 탭의 "인기 하이라이트"가 MVP 초기에 콘텐츠 부족으로 텅 빌 수 있음 | 초기에는 "최근 업로드" 기반 + 유저 수 100명 넘으면 "인기"로 전환. 빈 상태 대비 필요 |
| P-03 | 부모 계정의 "영상 업로드만 가능"이 실제로 어떤 UX인지 불명확 | 부모가 업로드하면 자녀 프로필의 클립 라이브러리에 추가됨. 자녀가 Featured/Top Clip은 직접 지정 |

### 🎨 [UX팀] 리뷰

**관점: "실제 사용자가 5분 안에 핵심 가치를 경험할 수 있는가?"**

**✅ 잘 된 것:**
- 영상 업로드를 프로필 안에 녹인 것 — 맥락 있는 행동 (빈 슬롯 "+" = 직관적).
- 완성도 가이드 — "다음에 뭘 해야 하지?" 해소.
- 온보딩이 3스텝으로 간결.

**⚠️ 문제점 + 개선:**

| # | 문제 | 개선안 |
|---|------|--------|
| U-01 | 프로필 보기모드에서 "편집"으로의 전환이 명확하지 않음. LinkedIn은 연필 아이콘이 각 섹션마다 있음 | **각 섹션 우측 상단에 "편집" 아이콘** 배치. 전체 편집모드 진입이 아니라 **인라인 편집** (섹션별 진입) |
| U-02 | Featured 슬롯이 1~3개인데, 최초에 빈 슬롯 3개가 보이면 "채워야 할 것"이 많아 보여 부담 | **처음에는 슬롯 1개만 보여주고**, 1개 채우면 2번째 슬롯 등장 (Progressive Disclosure). 최대 3개까지 |
| U-03 | 하이라이트 생성 1~2분 대기 시간 동안 사용자가 이탈할 수 있음 | 대기 중 **"기다리는 동안 스탯 기록해볼까요?"** 같은 크로스 액션 유도. 완료 시 푸시 알림 |
| U-04 | 프로필이 길어지면 스크롤이 과해짐 (Featured+Stats+Tags 6개+Season) | **프로필 상단: 핵심 요약** (카드+Featured+핵심 스탯). 하단: 상세 (태그 전체+시즌). 또는 **접히는 섹션(Accordion)** 활용 |
| U-05 | 태그별 Top Clip 지정이 별도 액션인데, 첫 클립 업로드 시 자동 Top Clip 설정이 자연스러움 | **해당 태그의 첫 번째 클립은 자동으로 Top Clip**. 이후 변경은 수동 |
| U-06 | 팀 앨범 "가져오기" 시 하이라이트 생성 체크박스 — 기본 체크로 해두면 불필요한 하이라이트가 쌓일 수 있음 | 기본값을 **"하이라이트 생성 안 함"**으로 변경. 사용자가 원할 때만 체크 |

### 💼 [사업팀] 리뷰

**관점: "사용자 획득과 리텐션이 가능한 구조인가? 수익화 여지가 있는가?"**

**✅ 잘 된 것:**
- 탐색 탭 = 콘텐츠 소비 공간 → DAU 확보 장치.
- "행동 → 피드 자동 게시" → 바이럴 루프 (A가 올리면 B가 보고 자기도 올림).
- 하이라이트 생성 = 명확한 과금 포인트 (후일).

**⚠️ 문제점 + 개선:**

| # | 문제 | 개선안 |
|---|------|--------|
| B-01 | 초기 유입(Acquisition)이 결국 "팀 코치가 소개"에 의존하는데, 코치를 설득할 키 메시지가 설계에 없음 | 팀 페이지 자체가 **"우리 팀 선수들을 한눈에"** 쇼케이스가 되어야 함. 팀 프로필 공유 링크 (/t/{team_handle}) 제공 → 코치가 학부모에게 공유 |
| B-02 | 공유(Share)가 프로필에만 있는데, 하이라이트 개별 공유도 중요 (카톡/인스타 스토리 등) | **하이라이트 개별 공유 링크 + 소셜 공유시트** 추가. 특히 카카오톡 공유 시 OG 이미지 최적화 |
| B-03 | 스카우터 경험이 "열람"에 그침. 관심 표현 방법이 없음 (DM은 v2) | v1 대안: **프로필에 "연락처 공개 설정"** (이메일 또는 전화번호를 선택적으로 노출). 스카우터가 직접 연락 가능 |
| B-04 | 팀 공개 프로필 (/t/{handle})이 있으면 초기 GTM에 강력한 도구가 됨 | **팀 공개 프로필 URL 추가** — 로고+소개+멤버 프로필 리스트 → 코치가 학부모/스카우터에게 공유 |

### 👨‍👩‍👦 [학부모] 리뷰

**관점: "내가 실제로 쓸 때, 불편하지 않은가? 아이에게 도움이 되는가?"**

**✅ 잘 된 것:**
- 부모 역할이 명확 (열람+업로드). 아이 대신 편집하는 갈등 없음.
- 카카오톡 SSO — 별도 회원가입 필요 없어서 좋음.
- 프로필 완성도 가이드 — 아이에게 "여기 이거 채워" 하고 알려줄 수 있음.

**⚠️ 문제점 + 개선:**

| # | 문제 | 개선안 |
|---|------|--------|
| PR-01 | 부모가 촬영한 영상을 어디서 올리는지 모호. "내 프로필"이 없으니 진입점이 없음 | 부모 홈에서 **자녀 프로필 바로가기** (상단 고정). 자녀 프로필에서 "영상 올려주기" 별도 버튼 |
| PR-02 | 경기 당일, 경기장에서 촬영하고 바로 올리고 싶은데 플로우가 4단계임 | **빠른 업로드**: 부모용 간소화 플로우 — 영상 선택 → 태그 선택 → 끝 (메모 생략, 하이라이트는 나중에 자녀가 결정) |
| PR-03 | 부모끼리 소통할 공간이 없음 | v1에서는 의도적으로 제외. 카톡 단체방이 이미 존재. v2+에서 팀 내 부모 커뮤니티 검토 |

### 13. Claude Code 개발 시 참고사항

이 설계서를 기반으로 개발 시 주의점:

**네비게이션:**
- 4탭 바텀: 홈(Feed) / 탐색(Discover) / 프로필(Profile) / 팀(Team)
- 설정은 프로필 탭 ⚙ 아이콘으로 접근

**프로필:**
- 요약 영역(상단) + 상세 영역(하단) 2단 구조
- 각 섹션 인라인 편집 (전체 편집모드 X)
- Featured: Progressive Disclosure (1→2→3개)
- Tags: Accordion (접기/펼치기)

**업로드 플로우:**
- 별도 탭 없음. 프로필 내 진입점 3곳
- 영상선택 → 태그+메모(합침) → 업로드+생성 → 리뷰
- 대기 중 크로스 액션 유도

**팀:**
- 2탭 (멤버/앨범)
- 멤버 카드 = 사진+이름+포지션+나이+메달수+레벨
- 누구나 생성, 관리자 권한 공유

**소셜:**
- 응원(Kudos) = 좋아요 대신. 아이콘 👏
- 피드 카드 6종 자동 생성

**기타:**
- 카카오 SSO
- 영상 최대 5분, 720p
- OG 이미지 최적화 (카카오톡 공유)
