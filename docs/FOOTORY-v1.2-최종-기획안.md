## 🧭 대전제 — "개인이 먼저, 팀은 나중에"

Footory는 팀 관리 앱이 아니다.
Hudl이나 TeamSnap처럼 팀 제휴(B2B)로 크는 모델이 아니라,
**개인 유소년 선수가 자기 축구 인생을 기록하고, 소통하고, 즐기는 앱**이다.

```
우선순위:
1순위: 개인 프로필 ← 앱의 심장
2순위: 소셜 (DM, 응원) ← 리텐션 엔진
3순위: MVP 경쟁 ← 재미 + 재방문
4순위: 탐색/검색 ← 발견
5순위: 팀 ← 소속 라벨 + 팀 피드
```

팀은 "소속"이지 "목적"이 아니다.
나중에 이 기록들이 진학, 스카우팅, 커리어의 포트폴리오가 된다.

### 한국 유소년 축구 현실 — 왜 이게 중요한가

```
스카우팅:
  - U-15 스카우트가 초5~6학년 때 시작 ("콜")
  - 스카우터가 영상 관리할 도구가 없음 (카톡+엑셀)
  - Footory 프로필 = 스카우터의 1차 필터

부모:
  - 자녀 영상을 유튜브 비공개에 올리는 게 전부
  - 진학 시 "영상 보내줘" → 체계적 포트폴리오 없음
  - USB에 영상 담아서 감독님께 전달하는 현실

선수:
  - 초등축구부 → 중학(U-15 유스 or 학교축구부) → 고등(U-18 유스 or 명문고)
  - K리그 입단 확률: 초등부터 시작해 상위 0.78%
  - 그래도 기록은 남아야 함 → 특기생 입시, 체육 관련 진학에 활용

→ Footory의 프로필 PDF, 성장 타임라인, 공개 프로필이
  현실적으로 가치를 만드는 포인트
```

## PART 1. 네비게이션 재설계

### 바텀탭 변경

```
v1.1: 🏠홈    🏆MVP    🔍탐색    👤프로필    👥팀
v1.2: 🏠홈    🏆MVP    💬DM     👤프로필    👥팀

라우트:
  /         → 홈 (추천 피드)
  /mvp      → MVP (주간 투표)
  /dm       → DM (대화 목록)    ← 신규
  /profile  → 프로필
  /team     → 팀
```

**왜 DM을 탭으로 올렸나:**
- 인스타(5탭), 틱톡(메시지=탭), 트위터(메시지=탭), 카카오(채팅=1번탭)
- 소셜이 핵심인 앱은 대부분 메시지를 바텀탭에 넣음
- 헤더 아이콘으로는 매일 누르지 않음 → 탭이면 매일 확인

**탐색은 어디로?:**
- 헤더 🔍 아이콘 → 검색+탐색 오버레이 (풀스크린)
- 기존 탐색 탭의 모든 콘텐츠 유지 (랭킹, 검색, 태그)
- 인스타의 검색/탐색 탭과 동일한 패턴

### 헤더 변경

```
v1.1: FOOTORY                    🔔

🔍 탭 → 검색+탐색 오버레이
```

## PART 2. 계정 관리 — "쉬운 진입, 점진적 검증"

### 원칙

```
1. 가입은 최대한 쉽게 (카카오 1탭)
2. 역할 구분은 온보딩에서 자기 선언
3. 검증은 팀 소속 + 활동으로 간접 확인
4. 코치/스카우터만 추가 인증 (DM 권한 때문에)
5. 14세 미만 부모 동의: v1.3으로 연기 (법적 검토 후)
```

### 계정 유형 3종

```
⚽ 선수 (기본)
  - 프로필: 선수 카드 (EA FC 스타일)
  - 할 수 있는 것: 영상 업로드, MVP 투표, 팔로우, DM, 응원/댓글
  - 동기: 자랑, MVP, 프로필 꾸미기, 진학 포트폴리오

👨‍👩‍👦 부모/보호자
  - 프로필: 없음 (자녀 뷰)
  - 할 수 있는 것: 자녀 활동 확인, 대신 업로드, 코치에게 DM, PDF 내보내기
  - 동기: 아이 잘하는지 확인, 영상 정리, 진학 준비

📋 코치/스카우터
  - 프로필: 코치 프로필 (✅ 인증 뱃지)
  - 할 수 있는 것: 팀 관리, 코치 태그/평가, DM (인증 후), 워치리스트(스카우터)
  - 동기: 선수 피드백, 스카우팅, 부모 소통
```

### 온보딩 플로우

**공통 Step 1:** 카카오 로그인 → 역할 선택 화면

```
"어떤 분이세요?"
  ⚽ 선수 — "직접 축구하는 선수예요"
  👨‍👩‍👦 부모/보호자 — "자녀의 축구를 응원해요"
  📋 코치/스카우터 — "선수를 지도하거나 발굴해요"
```

**선수 온보딩:**
```
이름/포지션/출생연도 → 키/몸무게/선호발(선택) → 팀 가입(선택) → 프사(선택) → 홈
목표: 30초 내 완료
```

**부모 온보딩:**
```
자녀 연결(검색/초대/나중에) → 완료 → 자녀 대시보드
자녀 연결: 자녀에게 연결 요청 알림 → 수락 시 parent_links 생성
```

**코치/스카우터 온보딩:**
```
이름/소속/역할상세 → 인증(선택: 팀코드/증빙/추천) → 팀 관리(선택) → 완료
미인증으로도 앱 사용 가능, 미성년 DM만 불가
```

### 코치 인증 3가지 방법

```
① 팀 초대 코드 입력 → 팀 admin 확인 → 즉시 인증
② 소속 증빙 제출 (자격증/명함) → 운영팀 검토 24~48h
③ 기존 인증 코치 추천 → 즉시 인증

인증 완료 시: 프로필 ✅ 뱃지 + 미성년 DM 가능
```

## PART 4. DM / 메시징

### 왜 DM인가

```
지금의 빈틈:
  응원만 누르면 아쉬움 → "그 돌파 어떻게 한 거야?" 물어볼 곳 없음
  코치가 선수에게 연락 → 카톡 번호를 알아야 함
  팀 공지 → 카톡 단톡방에서 → Footory 밖에서 일어남

DM이 만드는 가치:
  약한 연결(팔로우) → 강한 연결(DM) → 리텐션
  코치/스카우터 → 선수 직접 연락 → 실용적 가치
```

### v1.2 범위

| 기능 | v1.2 | v1.5 |
|------|------|------|
| 1:1 텍스트 DM | ✅ | ✅ |
| 영상 클립 공유 | ✅ | ✅ |
| 읽음 표시 ✓✓ | ✅ | ✅ |
| 메시지 삭제 | ✅ | ✅ |
| 차단/신고 | ✅ | ✅ |
| DM 요청 (비팔로우) | ✅ | ✅ |
| 그룹 DM (팀 단톡) | ❌ | ✅ |
| 이미지 첨부 | ❌ | ✅ |

## PART 9. 팀 미니멀 재설계

### 팀 = 소속 라벨

```
기존: 팀 = 관리 단위 (앨범, 공지, 멤버 관리...)
변경: 팀 = 프로필의 소속 태그 + 팀원들의 자동 피드

"나는 수원FC U-15야" → 프로필에 표시
"이전에 강남FC U-14였어" → 이력에 기록
"팀 동료들이랑 연결돼 있어" → 소셜 네트워크
```

### 변경 사항

```
팀 앨범 → 삭제. 멤버들의 영상이 "팀 피드"로 자동 표시
팀 관리 → 최소화. 팀 코드로 가입, 관리자가 강퇴만 가능
팀 코드 → 신규. 팀 생성 시 자동 생성, 카톡으로 공유
```
