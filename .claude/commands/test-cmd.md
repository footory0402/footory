전체 앱을 테스트해줘.

## 1. 빌드
- npm run build 실행
- npx tsc --noEmit 실행

## 2. 화면별 렌더링
각 페이지가 에러 없이 렌더링되는지:
/ (홈), /discover, /profile, /team, /p/[handle], /t/[handle]

## 3. 기능 플로우 확인
코드 분석해서 구현 여부 체크:
- 신규유저: 로그인 > 온보딩 > 프로필생성 > 레벨1
- 영상: 파일선택 > 용량체크 > 태그 > R2업로드 > DB저장
- 프로필: Featured > 스탯입력 > 메달부여 > 레벨업
- 소셜: 팔로우 > 피드 > 응원 > 댓글

## 4. 에러 처리
네트워크에러 UI, 빈상태 UI, 로딩, 업로드실패 재시도

## 5. 반응형
375px 430px 레이아웃 코드 분석.
